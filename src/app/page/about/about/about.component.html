ToDo App

Todo app is an easy to use online Todo list. It helps you to get organized, simplify your life, and to get things done.

You can perform following with this application,

  • Add a ToDo
  • Edit a ToDo
  • Toggle a ToDo, to mark as completed or pending
  • Delete a ToDo

This is fast and fluid because it use browser local storage to store your todo list.

Note: Beware before cleanning your browser local storage.
